Cartegraph Requests

  1. Requests are created when a customer (citizen, co-worker, partner, contractor, etc.) requests a service, such as inspection, repair, installation, etc.
  2. Next, a Task or a group of Tasks is created to address a Request. If a Request requires multiple Tasks, more Tasks can be added to a Request.Â 
  3. Further, a Task or a group of Tasks goes into a Work Order.

What is a Request

Request is a formal register of a customer-reported issue, complaint, request, order. Request is the CSR precursor.
